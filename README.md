# POKEMON
LANDING PAGE OF POKEMON
A visually striking, minimalist landing page dedicated to the most iconic Pokémon, Pikachu. This project highlights a split-screen layout and modern web design principles to create a premium fan experience.

⚡ Project Spotlight
The design focuses on a "Hero" section that utilizes a two-tone background to create depth. It combines large, impactful typography with character-driven content, making it an excellent example of a modern Single Page Application (SPA) landing screen.

✨ Key Features
Dual-Tone Layout: A dynamic vertical split background that highlights the character artwork.

Typography Driven: Use of bold, heavy fonts for the "Pikachu" heading to establish a clear visual hierarchy.

Vertical Navigation: A sleek, minimal side-pagination slider indicating the current view (01 / 08).

Watermark Background: A large, low-opacity Pokéball icon in the background for added brand texture.

Responsive Header: A clean navigation bar featuring the official logo and a functional hamburger menu.

🛠️ Tech Stack
HTML5: Structured for high accessibility and SEO.

CSS3: Utilizing Flexbox for alignment and absolute positioning for the background elements.

Google Fonts: Integrated high-weight sans-serif fonts for that "bold" look.

Images: High-resolution transparent PNGs to achieve the "overlapping" character effect.
